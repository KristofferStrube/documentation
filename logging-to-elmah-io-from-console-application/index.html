ï»¿# Logging from Console

Even though elmah.io support various logging frameworks like Serilog, log4net and NLog, logging from a simple console application is dead simple.

To start logging, install the elmah.io.client NuGet package:

Install-Package elmah.io.client

Create a new Loggerand assign it to a variable of type ILogger:

Elmah.Io.Client.ILogger logger = new Elmah.Io.Client.Logger(new Guid("LOGID"));

Replace LOGID with your log ID from elmah.io.

The elmah.io client supports logging in different log levels much like other logging frameworks for .NET:

logger.Verbose("Verbose message");
logger.Debug("Debug message");
logger.Information("Information message");
logger.Warning("Warning message");
logger.Error("Error message");
logger.Fatal("Fatal message");
